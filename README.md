## How to use this

Call any agent **by name** in any Claude session — chat, Code, Cowork, or the Telegram runtime. They automatically inherit:

1. **Their role** from the protocol `makabi-os-agent-roster` in Makabi Memory
2. **Their tools** from the Makabi Agent OS (113 tools across Stripe, GHL, Supabase, GitHub, GoDaddy, Retell, ElevenLabs, HeyGen)
3. **Their brand context** from the active repo or client (1PM / BRiX / Loren AI / MTIP / etc.)
4. **Their write-back discipline** from the `mandatory-memory-discipline` protocol — every session ends with a Memory write

## Escalation path

```
Specialist  →  Department Head  →  MASSIMO  →  Master Makabi
```

Never skip rungs. Massimo is the single throat to choke and the single entry point for delegation in either direction.
